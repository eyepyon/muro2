# ムッシュムロムロ

 日本古来から伝わる「雪室」「氷室」を活用したReFiサービス。 
 自然のクーラーといわれる雪室をうまく使うことでCo2が大幅に削減できます。 
 近年豪雪被害に悩まされる地方の厄介者である雪を保管し活用時に評価し 
 カーボンクレジットとして活かすReFiサービス。
